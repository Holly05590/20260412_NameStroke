# 姓名學筆劃鑑定

中文姓名學筆劃計算工具，支援即時筆劃查詢、五格剖象分析、動力元素計算與流年運程圖。  
純前端，雙擊 `index.html` 即可在瀏覽器使用，無需伺服器。

---

## 功能

- 輸入姓氏、名一（必填）、名二（選填），即時顯示每字筆劃數
- 依康熙字典繁體筆劃標準，套用姓名學偏旁補正規則
- 自動計算五格剖象（宮位制）：父母宮、疾惡宮、奴僕宮、遷移宮、命宮
- 顯示三才五行（父母宮、疾惡宮、奴僕宮對應五行屬性）及相鄰五行生剋徽章
- 自動計算動力元素（疾惡宮與奴僕宮的數字差縮減值）
- 支援二字名（姓＋名一）及三字名（姓＋名一＋名二）
- 支援倉頡、注音、拼音等輸入法（compositionend 事件處理）
- 清空按鈕一鍵重置所有輸入（含出生日期）
- 三種外觀主題可切換，選擇自動記憶
- **流年運程圖**：顯示十二長生 10 階段圓形圖，含流年數字標註、年齡徽章、當年流年格高亮

---

## 使用方式

1. 開啟 `index.html`（直接雙擊或拖入瀏覽器）
2. 輸入姓氏與名一（各一個中文字）；名二可留空（二字名）
3. 輸入完成後自動計算並顯示結果
4. 在流年運程圖上方輸入出生年月日（民國年），即時顯示年齡並高亮當年流年格
5. 點擊「清　空」按鈕可重新輸入（姓名與出生日期全部清空）
6. 點擊標題下方的色點按鈕可切換主題風格

### 筆劃資料來源

採用三階段查詢：

1. **STROKE_DB**（優先）：手工校正的常用姓名字資料庫，確定正確的字直接給出結果
2. **Unihan kRSUnicode**（主要退回）：Unicode 官方資料庫，28,000+ 個漢字，以「康熙部首全形筆劃 + 剩餘筆劃」計算，自動涵蓋所有偏旁補正，無需手動 RADICAL_DELTA
3. **cnchar-all**（最終退回）：處理極少數 Unihan 未收錄的罕見字

## 五格剖象公式

### 三字名（姓＋名一＋名二）

| 宮位 | 原稱 | 公式 | 含義 |
|------|------|------|------|
| 父母宮 | 天格 | 姓筆劃 ＋ 1 | 祖先遺傳、先天運 |
| 疾惡宮 | 人格 | 姓筆劃 ＋ 名一筆劃 | 個性、主運 |
| 奴僕宮 | 地格 | 名一筆劃 ＋ 名二筆劃 | 後天運、基礎 |
| 遷移宮 | 外格 | 名二筆劃 ＋ 1 | 社交、對外關係 |
| 命 宮 | 總格 | 姓 ＋ 名一 ＋ 名二筆劃合計 | 整體命運 |

### 二字名（姓＋名一，名二空白時自動套用）

| 宮位 | 公式 | 說明 |
|------|------|------|
| 父母宮 | 姓 ＋ 1 | 同三字名 |
| 疾惡宮 | 姓 ＋ 名一 | 同三字名 |
| 奴僕宮 | 名一 ＋ 1 | 虛數補 1 |
| 遷移宮 | 1 | 虛數固定 1 |
| 命 宮 | 姓 ＋ 名一 | 僅計實字 |

### 驗證範例：王大明（三字名）

| 字 | 標準劃數 | 補正 | 姓名學劃數 |
|----|--------|------|-----------|
| 王 | 4 | — | 4 |
| 大 | 3 | — | 3 |
| 明 | 8 | — | 8 |

→ 父母宮 **5**、疾惡宮 **7**、奴僕宮 **11**、遷移宮 **9**、命宮 **15**

### 驗證範例：陳○○（含阝左補正）

陳：標準 11 劃，阝左（阜旁）補正 +5 → 姓名學 **16** 劃

## 動力元素

動力元素反映疾惡宮與奴僕宮之間的數字張力，結果為 0–4。

### 計算公式

1. 取疾惡宮數字的十位數＋個位數 = **a**
2. 取奴僕宮數字的十位數＋個位數 = **b**
3. 差數 = |a − b|
4. 若差數 > 4，則反覆以 **9 − 差數** 取代，直到 ≤ 4

### 計算範例

```
疾惡宮 = 18 → 1+8 = 9（a）
奴僕宮 = 13 → 1+3 = 4（b）
差數 = |9−4| = 5 → 大於 4 → 9−5 = 4
動力元素 = 4
```

## 三才五行對應

父母宮、疾惡宮、奴僕宮的個位數對應五行：

| 個位數 | 五行 |
|--------|------|
| 1、2 | 木 🌿 |
| 3、4 | 火 🔥 |
| 5、6 | 土 ⛰ |
| 7、8 | 金 ✦ |
| 9、0 | 水 💧 |

### 五行關係徽章

五行欄位之間顯示圓形徽章，以箭頭符號說明相鄰宮位的五行生剋關係：

| 符號 | 類型 | 說明 |
|------|------|------|
| ▽ | 相生（向下） | 上格之五行生下格 |
| △ | 被生（向上） | 下格之五行生上格 |
| ▼ | 相剋（向下） | 上格之五行克下格 |
| ▲ | 受剋（向上） | 下格之五行克上格 |
| ‖ | 比和 | 兩格五行相同 |

徽章以絕對定位顯示於兩格正中間，視窗縮放時自動重新定位。

## 流年運程圖

### 圖形結構

以 SVG 動態產生的圓形命盤，顯示傳統命理「十二長生」10 個宮位階段：

| 宮位 | 中心角度 | 意義簡述 |
|------|--------|---------|
| 胎 | 0° | 入胎受氣 |
| 養 | 36° | 培育滋養 |
| 長生 | 72° | 氣象初生 |
| 冠帶 | 108° | 進學成長 |
| 臨官 | 144° | 入仕掌事 |
| 帝旺 | 180° | 鼎盛極旺 |
| 衰 | 216° | 氣勢漸退 |
| 病 | 252° | 精力衰弱 |
| 死 | 288° | 氣絕沉寂 |
| 絕 | 324° | 絕跡待生 |

- 中央為 4 芒羅盤星形，顏色使用 `--accent` 主題色
- 圓圈外側標示每段起止角度（0°～342°，每段 36°）
- 所有顏色使用 CSS 變數，自動配合三種主題

### 流年數字標註

輸入姓名並完成計算後，以**命宮個位數**為起始，從冠帶開始依序標註 0–9 於各宮位外環：

```
起始位置：冠帶
公式：各宮位數字 = (命宮個位數 + 該宮距冠帶偏移量) mod 10
```

**範例：命宮 32（個位數 2）**

| 宮位 | 數字 |
|------|------|
| 冠帶 | 2 |
| 臨官 | 3 |
| 帝旺 | 4 |
| 衰 | 5 |
| 病 | 6 |
| 死 | 7 |
| 絕 | 8 |
| 胎 | 9 |
| 養 | 0 |
| 長生 | 1 |

### 出生日期與年齡

流年圖上方提供出生日期輸入欄（民國年、月、日）：

- **年齡計算採虛歲制**：出生那年算 1 歲
- **前後半年規則**：每年「生日後半年」（生日月份 +6 個月的同一天）為歲數切換點；切換點起歲數加 1
- **年齡徽章**：計算出的虛歲顯示於流年圖右上角（主色系徽章）
- **當年流年格高亮**：年齡個位數與哪個宮位的流年數字相同，該格即以主色填色標示

#### 虛歲計算範例

```
出生：民國 62 年 4 月 5 日
切換點：每年 10 月 5 日（4月 + 6個月）

民國 114 年 10 月 5 日起 → 虛歲 = 115 − 62 + 1 = 54
民國 115 年 10 月 4 日止 → 仍為 54 歲
民國 115 年 10 月 5 日起 → 虛歲 = 116 − 62 + 1 = 55
```

清空輸入後，數字、年齡徽章及高亮扇形自動隱藏。

## 已知限制

- 僅支援單姓格式（複姓五格公式不同，本工具未涵蓋）
- 偏旁補正表依常用姓名字建立，罕見字可能無補正
- 各姓名學門派計算規則略有出入，結果僅供參考
- 虛歲切換點以西曆月日比較，不採農曆；閏月、月底等邊界情況可能有 1 日誤差
